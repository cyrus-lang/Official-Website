# نحو پایه

سایروس دارای نحو ساده و قابل فهمی است که یادگیری آن آسان است.

## توضیحات

```cyrus
// این یک توضیح تک خطی است
/* این یک توضیح چند خطی است */
```

## ثابت‌ها

```cyrus
// رشته‌ها
let message = "سلام دنیا";

// اعداد
let number = 42;
let float = 3.14;

// بولین
let is_true = true;
let is_false = false;
```

## کاراکترهای فرار

```cyrus
let newline = "\n";
let tab = "\t";
let quote = "\"";
```

## انواع پایه

```cyrus
let integer: i32 = 42;
let float: f64 = 3.14;
let boolean: bool = true;
let string: str = "سلام";
```

## ثابت‌های عددی

```cyrus
let binary = 0b1010;      // دودویی
let hex = 0xFF;           // هگزادسیمال
let octal = 0o77;         // هشت‌تایی
```

## آرایه‌ها

```cyrus
let numbers = [1, 2, 3, 4, 5];
let first = numbers[0];
```

## بردارها

```cyrus
let mut vector = [1, 2, 3];
vector.push(4);
```

## اشاره‌گرها

```cyrus
let value = 42;
let pointer = &value;
```
