# متغیرها

متغیرها در سایروس برای ذخیره داده‌ها استفاده می‌شوند.

#### تعریف متغیر

```cyrus
let name = "احمد";
let age = 25;
let height = 1.75;
```


####  متغیرهای قابل تغییر

```cyrus
let mut counter = 0;
counter = counter + 1;
```


####  انواع صریح

```cyrus
let name: str = "احمد";
let age: i32 = 25;
let is_student: bool = true;
```


####  ثابت‌ها

```cyrus
const PI = 3.14159;
const MAX_SIZE = 100;
```
