# توابع

توابع در سایروس برای گروه‌بندی کد و قابلیت استفاده مجدد استفاده می‌شوند.

## تعریف تابع ساده

```cyrus
fn greet() {
    println("سلام!");
}
```

## تابع با پارامتر

```cyrus
fn greet(name: str) {
    println("سلام " + name + "!");
}
```

## تابع با مقدار بازگشتی

```cyrus
fn add(a: i32, b: i32) -> i32 {
    return a + b;
}
```

## تابع با مقدار بازگشتی ضمنی

```cyrus
fn multiply(a: i32, b: i32) -> i32 {
    a * b  // آخرین عبارت به عنوان مقدار بازگشتی در نظر گرفته می‌شود
}
```

## فراخوانی توابع

```cyrus
fn main() {
    greet();
    greet("احمد");

    let result = add(5, 3);
    println("نتیجه: " + result);
}
```
