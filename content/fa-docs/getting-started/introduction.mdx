# مستندات

به مستندات رسمی کوروش خوش آمدید. کوروش یک زبان برنامه‌نویسی سیستمی برای ساخت زیرساخت‌هایی است که باید سال‌ها کار کنند — آشکار درباره هر تصمیم در سطح ماشین، بدون هزینه‌های پنهان، runtime، یا نگهبانی کامپایلر.

## یک نمونه سریع

در اینجا یک نمای اجمالی از کد کوروش را می‌بینید:

```cyrus
import std::libc{printf};

pub fn main() {
  const msg: char* = "Cyrus";

  printf("Hello, %s!\n", msg);
}
```
