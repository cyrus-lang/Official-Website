# سلام کوروش!

بیایید با اولین برنامه شروع کنیم:

```cyrus
import std::libc;

pub fn main() {
    libc::printf("Hello, Cyrus\n");
}
```

## کامپایل کردن برنامه

برنامه بالا را در فایلی به نام **hello_world.cyrus** قرار می‌دهیم.

سپس می‌توانیم آن را با دستور زیر اجرا کنیم:

```bash
cyrus run hello_world.cyrus
```

خروجی باید `Hello, Cyrus!` را چاپ کرده و به خط فرمان بازگردد.
