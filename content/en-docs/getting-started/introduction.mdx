# Documentation

Welcome to the official documentation for Cyrus. Cyrus is a systems programming language for building infrastructure that must run for years — explicit about every machine-level decision, without hidden costs, runtime, or compiler guardianship.

## A Quick Taste

Here's a glimpse of what Cyrus code looks like:

```cyrus
import std::libc{printf};

pub fn main() {
  const msg: char* = "Cyrus";

  printf("Hello, %s!\n", msg);
}
```
